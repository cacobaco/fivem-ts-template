# fivem-ts-template

A basic typescript template for fivem scripts.

## Usage

Clone the project

```bash
  git clone https://github.com/cacobaco/fivem-ts-template
```

Install the dependencies

```bash
  npm i
```

Build

```bash
  npm run build
```

Or build automatically (detects file changes)

```bash
  npm run watch
```
